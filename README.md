# Ročníkový projekt - Simple Puzzle Game

# Námět

Rozhodl jsem se vytvořit jednoduchou 3D hru v Unreal Enginu 5, ve které z pohledu první osoby má hráč za úkol vyřešit jak se dostat z místních katakomb.

# Stav Projektu

Projekt je v tuto chvíly hotový, spolu s interaktivním prostředím a mechaniky splnění daných úkolů.
